个人概况

本科期间做的和发起的项目：

  • 一个实现了拼单、拼车、群聊等功能的 Android App （java）

研究生期间的方向很乱，包括：

尽管如此，本人不变的特性是喜欢 coding ，有代码审美，喜欢折腾并且经验丰富（比如自己修改 Linux 内核参数编译），愿意花时间在提高效率上。
