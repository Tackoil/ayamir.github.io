在 Jupyter Notebook 中设置 Conda 环境

Ayamir 出版于 Development
 2022-02-15  约 129 字   预计阅读 1 分钟 
目录

  1. 远程启动jupyter notebool：

    1

    jupyter notebook --no-browser --ip="<server-ip>" --port="<server-port>"

  2. 激活预先配置好的conda环境，这里假设环境名为keras-tf-2.1.0：

    1

    conda activate keras-tf-2.1.0

  3. 安装ipykernel：

    1

    pip3 install ipykernel --user

  4. 为ipykernel安装环境：

    1

    python3 -m ipykernel install --user --name=keras-tf-2.1.0

  5. 打开notebook更改服务之后刷新即可：
