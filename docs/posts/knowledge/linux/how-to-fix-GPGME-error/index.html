修复 Archlinux 上出现的 GPGME Error

Ayamir 出版于 Knowledge
 2021-06-11  约 32 字   预计阅读 1 分钟 
目录
    • Delete old sync files
    • Re init pacman-key
    • Populate key
    • Re sync

Delete old sync files

1

sudo rm /var/lib/pacman/sync/*

Re init pacman-key

1

sudo pacman-key --init

Populate key

1

sudo pacman-key --populate

Re sync

1

sudo pacman -Syyy

Now you can update successfully!
